This page loads https://gpay-live-demo.web.app in an <iframe>.
The demo calls isReadyToPay() to determine if a Google Pay button should be displayed or not.

Because the iframe does not set the allow="payment" attribute, the Payment Request API is not available and the following happens:

  • Web: Button is shown and Google Pay falls back to POPUP mode because the PaymentRequest API is not available and therefore can't trigger the Payment Handler window.
  • Mobile Web (Chrome on Android): No button is displayed at all because bridging to the native isReadyToPay() API also needs the Payment Request API.
